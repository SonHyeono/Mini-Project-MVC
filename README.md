## Java Mini Project

프로젝트 주제 :
ETF를 사고 파는 프로그램입니다.

DTO :

1. ETF
2. ETFCustomer
3. ETFManager
4. ETFProject

MVC 패턴을 이용해서 StartView로 시작을 하여 ETFProjectController를 이용해서 Model들에 접근을 하고 메소드를 이용을 하는 패턴으로 만들었습니다.

다음은 프로젝트 개발 후에 3개의 Task를 수행을 하였습니다.

- Task 1

  - 예외처리
  - 새로운 메소드 만들기

- Task 2

  - lombok으로 코드단순화
  - toString 재정의해서 객체 이쁘게 보이기
  - Exception 통일하기.

- Task 3
  - lambda로 문법 전환
  - log4j를 추가
